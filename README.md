# IMChat

>
>即时通讯项目，支持好友、单聊、群聊

# 项目构成

```
api:用于定义接口函数 
cache:redis操作 
conf:初始化配置以及存放配置信息 
middlware：应用中间件 
model:数据库模型与初始化 
pkg--e:错误处理类 
   --util:工具类 
router：路由组 
serializer:序列化函数包 
service：服务模块 
```

# 演示流程

- 单聊
- 群聊
- 添加好友
- 查看历史记录

- cache : 放置redis配置
- conf : 放置配置文件 
- model : 数据库模型
- pkg : 防止一些错误码
- router ： 路由模块
